设置主题，导入已有的主题设置

  直接点击菜单的file–>import settings…，选中你刚刚下载的那个jar包，然后在弹出的对话框中选中所有选项，然后点击ok重启webstorm之后就生效了，传说中的钢琴黑有没有惊艳到你呢，仅仅只是背景色嘛？nonono~代码高亮，警告左边会变成黄色，error直接变成红色，更改过的是蓝色，添加的是绿色。。。自己慢慢发现吧！当然，你也可以自己设置导出jar包，操作一样的。

Patch应用

  可能在某些时候由于一些不得已的原因，我们本地没法push代码到远程了，而刚好又有个紧急的任务需要用到这次push的代码内容，这个时候我们就可以用到patch了，它可以帮我们检出本次提交的代码内容，并形成一个.patch的文件。我们应用这个文件到别的项目中，就可以看到里面的代码内容了。

  当代码还没有提交的时候，我们可以直接创建patch：VCS–》create patch

  当代码以已经提交了的时候，我们需要找到那次提交日志，然后将这次提交的内容打成一个patch：依次点开编辑器最下面的version control面板，然后切换到log面板，里面有最近提交的日志信息，找到你需要创建patch的那次提交，右键，然后菜单中选择 create patch，如下：

生成完patch之后我们需要应用它，不然生成来干嘛呢。菜单栏VCS–》apply patch ，选择你刚刚生成的patch文件，apply一下就行了。
